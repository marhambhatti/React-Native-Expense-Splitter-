# 🚀 Expo React Native Application

A modern mobile application built using **React Native** and **Expo**. This project provides a scalable foundation for developing cross-platform applications that run seamlessly on Android, iOS, and Web.

---

## 📋 Overview

This project was created using **Expo** and follows a file-based routing architecture powered by Expo Router. It is designed to accelerate development by providing a clean project structure, fast refresh capabilities, and a streamlined development workflow.

---

## ✨ Features

- Cross-platform support (Android, iOS, Web)
- Expo Router with file-based navigation
- Fast Refresh for rapid development
- Modern React Native architecture
- Easy deployment using Expo tools
- Scalable folder structure
- Development and production build support

## ⚙️ Prerequisites

Before running the project, ensure you have:

- Node.js (Latest LTS Recommended)
- npm or yarn
- Expo CLI
- Android Studio (for Android Emulator)
- Xcode (for iOS Simulator on macOS)

---

## 🚀 Installation

### 1. Clone the Repository

```bash
git clone https://github.com/marhambhatti/React-Native-Expense-Splitter-.git
cd Expense_Spliter
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Start the Development Server

```bash
npx expo start
```

---

## 📱 Running the Application

After starting the development server, choose one of the following options:

### Android Emulator

```bash
a
```

### iOS Simulator

```bash
i
```

### Web Browser

```bash
w
```

### Expo Go

- Install Expo Go on your mobile device.
- Scan the QR code displayed in the terminal.

---

## 🔄 Reset Project

To create a fresh project structure:

```bash
npm run reset-project
```

This command moves starter code into the `app-example` directory and creates a clean `app` directory.
